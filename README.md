# TechStore - E-commerce Website

A simple e-commerce website for tech products, built from scratch in 2 hours without AI assistance.

## Project Overview

TechStore is a responsive e-commerce website showcasing various tech products including laptops, headphones, smartphones, tablets, smartwatches, and gaming keyboards. The project was completed as a practical exercise in web development.

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Styling with Flexbox and CSS Grid
- **Google Fonts** - Poppins font family

## Features

- Responsive navigation bar with logo and menu
- Product catalog with 6 products displayed in a grid
- Product cards with images, descriptions, and prices
- Monthly promotion section with discounted pricing
- Newsletter subscription form
- Footer with social media links
- Mobile-responsive design

## Project Structure

```
├── index.html       # Main HTML file
├── style.css       # CSS stylesheet
├── images/         # Product images and logo
│   ├── logo.png
│   ├── product1.jpg - product6.jpg
│   └── promo.jpg
└── README.md       # This file
```

## How to Run

1. Open the `index.html` file in any modern web browser
2. No server or build process required

## Time Spent

- Total development time: 2 hours
- No AI assistance used
